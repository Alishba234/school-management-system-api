# 📚 School Management API

A Node.js and Express-based REST API for managing school data.  
This system allows users to add new schools and retrieve a list of schools sorted by proximity using latitude and longitude.

---

## 🚀 Tech Stack

- Node.js (Runtime Environment)
- Express.js (Web Framework)
- MySQL (Database)
- Sequelize ORM (Database ORM)
- Joi (Input Validation)
